## How To's

| what                                | how                                   |
| ----------------------------------- | ------------------------------------- |
| show current IP address             | `ip addr show`                        |
| current IP address (smaller result) | `ip addr show | grep inet`            |
| check memory/cpu usage of device    | `htop` (press F6 to sort by a column) |
| disk usage stats                    | `df -h`                               |
